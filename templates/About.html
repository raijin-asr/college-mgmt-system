1.1 Background

College Management System(CMS) is type of management system where records of every student and teacher is kept in computerized way. It is designed to automate the existing manual system through computerized medium for student information management and teacher information management. There are options of faculty available under different provided categories which they can choose as per their interest and needs. Student details are maintained and viewed in user friendly interface by this system. With the CMS, any college can maintain records of student and teacher faster saving time and effort.

1.2 Features

The main features of this project are mentioned below:
  • UI: User friendly and clean interface with organized menu options.
  • Database: Use of database to store and manipulate the student data and teacher data.
  • Login Module: Login as a admin, login as a student and login as a teacher.
  • Security: Due to use of Login system, only authorized person can access the system.
  • Multi-mode: Admin mode, Student Mode and Teacher Mode are available.
  • Operation: Use of CRUD operation in student and teacher information management.


1.3 Applications

This project is made especially to every student who come to college to study different desired courses. There is need to record how many students has enrolled based on faculty chosen by them and record correct information. So, this project does all these operations easily and faster. It is online system, so people just need to internet to use this system. This system can be used in any college for maintaining student details, courses and faculty chosen by them and their personal information.
